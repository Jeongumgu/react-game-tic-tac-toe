# react-game-tic-tac-toe

Maximilian Schwarzmüller의 Udemy react 강의를 듣고 구현해본 틱택토 게임

틱택토란?

- 두 명이 번갈아가며 O와 X를 3×3 판에 써서 같은 글자를 가로, 세로, 혹은 대각선 상에 놓이도록 하는 놀이이다. m, n, k-게임으로, -게임이다. 아래 예시는 먼저 놓은 X가 이기는 경우이다.
